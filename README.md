# Fintech Transaction payments

Microservice handling payment processing, card validation,
and fund transfers for FinPay platform.

## Features
- Card payment processing (Visa, Mastercard, Amex)
- Real-time transaction verification
- Multi-currency support
- Fraud detection scoring
- PCI-DSS compliance target

## Security Scanning
Monitored by ExposureIQ for CVE vulnerabilities
and secure coding violations.
